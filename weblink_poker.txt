在 Ubuntu 下建立 RAM Disk 的方法如下：

使用终端命令行输入：sudo mkdir /mnt/ramdisk
创建文件系统：sudo mount -t tmpfs -o size=512M tmpfs /mnt/ramdisk
可以通过更改 size 参数来调整 RAM Disk 的大小。
重启后 RAM Disk 的内容会丢失，因为 RAM Disk 是使用 RAM 作为存储介质的。
注意：使用 RAM Disk 可以提高系统读写速度，但也会增加系统的内存使用量。
